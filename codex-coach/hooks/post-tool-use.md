# PostToolUse

Watch for waste signals:

- Same command or tool fails three times.
- Context grows rapidly without new evidence.
- Premium model is used for formatting, extraction, or cleanup.
- Tool output is large and only a small part is task-relevant.

Template:

```text
Codex Coach nudge: {signal}. Recommended next move: {action}. This should reduce burn without changing the target outcome.
```

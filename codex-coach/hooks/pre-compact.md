# PreCompact

Before compaction, preserve only the state needed to continue well.

Template:

```text
Codex Coach compact note: project={title}; goal={goal}; status={status}; estimate={expectedCredits} expected credits; actual={actualCreditsOrUnknown}; model pattern={modelPattern}; next move={nextMove}; budget mode={budgetMode}.
```

If the session has no active project, emit nothing.
